Proyectos

Paw2Go

Como proyecto final del Bootcamp de desarrollo de software web que más recientemente he concluido, se construyo una aplicación e-commerce llamada Paw2go Se trabajó desde el diseño, maquetación y programación de la parte del cliente y del servidor. Por la parte del Frontend utilizamos tecnologías como HTML para el contenido de la página, CSS y Bootstrap para la decoración y hacer más atractiva visualmente nuestra página, JavaScript para añadirle interactividad con los usuarios y por la parte del Backend, se utilizó el lenguaje de programación Java y con la ayuda de Spring Framework se realizó la APIRest para la comunicación entre nuestra página web y una base de datos construida con MySQL donde se guarda la información de nuestros productos.
